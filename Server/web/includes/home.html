Bienvenue !

Bienvenue sur le site web créé dans le cadre du projet PCD de 2ème année de TELECOM Nancy.
Ce projet a pour but de présenter une application client-serveur permettant de planifier des trajets dans le Grand Nancy, mais aussi d'interroger les horaires des différentes lignes de transport en commun.
Notre site vous permet de télécharger notre client JAVA et de vous inscrire pour profiter pleinement de notre application.
Bonne visite!
